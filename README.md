## üöÄ How to Run

1. Clone this repository:
   ```sh
   git clone https://github.com/fadhalshulhan/fadhalshulhan.github.io.git
   ```

2. Install dependencies:
   ```sh
   npm install
   ```

3. Run the development server:
   ```sh
   npm start
   ```

4. Build for production:
   ```sh
   npm run build
   ```
